## [0.4.0] - 2026-04-29

Initial public release as a HACS-installable plugin.

### Added

- Room name + icon header with configurable tap, double-tap, and hold actions
- Top-right, bottom-left, and bottom-right entity slots with per-entity active/inactive colors, custom icons, and value display
- Per-entity show-only-when-active toggle and active-attribute matching
- Configurable border radius per corner
- Visual editor for all options
- Domain defaults for light, switch, fan, climate, media_player, cover, lock, binary_sensor, camera, vacuum, person, and device_tracker
- HA theme-aware color palette across the editor
